## 五、开发者生态 (Developer & Open Source)（三模块布局：开源链接+部署指南+API 预览）

    - 描述：开源权重完全开放，支持 LoRA 二次微调（适配企业专属术语库），共建翻译生态。

2. **快速部署指南**
    - 核心方案：
        - 本地部署：Ollama 一键运行（命令：`ollama run translategemma:12b`），无需复杂配置
        - 企业部署：支持分布式推理、4-bit 量化（显存占用降低 70%，精度损失≤1%），适配 A100/H100 GPU 集群
        - 端侧部署：4B 模型支持智能手机 SoC、低功耗边缘计算设备，满足移动端场景需求

3. **API 预览（Python 示例）**
    - 代码块：
      ```python
      from transformers import AutoModelForCausalLM, AutoTokenizer
 
      # 加载模型（12B 平衡版）
      model_id = "google/translategemma-12b"
      tokenizer = AutoTokenizer.from_pretrained(model_id)
      model = AutoModelForCausalLM.from_pretrained(model_id, device_map="auto")
 
      # 翻译示例（法律文本）
      text = "双方应在本协议生效后 30 日内完成标的资产的交割"
      inputs = tokenizer(f"Translate to English: {text}", return_tensors="pt").to("cuda")
      outputs = model.generate(**inputs, max_new_tokens=200)
      translation = tokenizer.decode(outputs[0], skip_special_tokens=True)
      print(translation)
      ```
    - 提示：“支持批量翻译、异步任务，兼容 vLLM 快速推理，查看完整 API 文档 →”

## 六、常见问题 (FAQ)（折叠面板，默认展开前 3 个高频问题）
1. **TranslateGemma 是免费的吗？**
    - 答：注册即送 10,000 免费 Credits（1 Credit = 1 字符翻译），用完可升级付费套餐；开源模型可免费下载，本地部署无额外费用，适合个人及中小企业初期使用。

2. **它与 Google Translate、ChatGPT 翻译有何不同？**
    - 答：TranslateGemma 是 Google 专为翻译优化的模型（非通用 LLM），通过数十亿双语语料微调+行业数据适配，聚焦翻译核心任务；术语准确率、语境理解远超通用模型，且支持开源部署，数据更安全，无“机翻感”。

3. **我的翻译数据安全吗？**
    - 答：绝对安全！在线翻译数据加密传输，72 小时内自动删除，不用于模型训练；支持私有化部署，数据全程在企业防火墙内流转，完全符合数据出境合规要求，你的秘密永远是你的秘密。

4. **支持哪些语言？**
    - 答：覆盖 55+ 核心语种，包括中英、中日、中德、英法等主流语言对，以及冰岛语、斯瓦希里语等低资源语种，低资源语种翻译质量较基线模型提升 25-30%。

5. **如何保证专业术语的准确性？**
    - 答：模型在微调阶段融入了法律、医疗、电商、科技等多行业双语语料，支持企业上传自有术语库进行 LoRA 微调，训练专属“品牌翻译官”，确保行业术语无偏差。

6. **Credits 计费规则是什么？**
