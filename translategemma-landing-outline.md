- 支持多模态兼容

**5:1 交错注意力机制**
- 内存复杂度从 O(n²) 降至接近 O(n)
- 单张 H100 GPU 处理 128k Token
- 长文本处理效率翻倍

**256k Token 词表**
- Llama 2 词表的 8 倍
- 提升非英语语言编码效率
- 减少形态复杂语言的拆分损耗

#### 🎯 两阶段训练
**SFT + RL 强化学习**
- 融合 Gemini 2.5 生成的 43 亿 Token 数据
- 多奖励模型集成优化
- 保障翻译自然度与准确性

#### 🖼️ 多模态融合
**图像文本统一编码**
- 图像转化为 256 个 Token 嵌入
- 无需 OCR 预处理
- 直接翻译图像中的文本
